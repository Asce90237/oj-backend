
# Open Judge

基于Spring Cloud + Docker + RabbitMQ + Redis的编程题目在线评测系统。系统能够根据管理员预设的题目用例对用户提交的
代码进行执行和评测；系统中自主实现的代码沙箱可作为独立服务供其他开发者调用。

# 代码沙箱 
- 基于Docker实现隔离 （Args模式）
- 基于原生Java实现 （ACM模式）
